# Go Gin Quickstart

This is a simple starter project for building web applications with [Gin](https://github.com/gin-gonic/gin), a high-performance web framework for Go.  

## Features

- Connects to PostgreSQL using GORM
- Basic routing with Gin
- Example `Ping` endpoint
